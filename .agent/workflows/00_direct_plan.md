1. **Load Context**: `.agent/agents/director.md` と `.agent/rules.md` を読み込む。
2. **Analyze**: ユーザーのリクエストを読む。
3. **Document**: `docs/requirements.md` を更新する。
4. **Plan**: アーキテクチャの決定事項を `docs/decisions.md` に作成/更新する。
5. **Delegate**: 次のステップを Researcher または Coder に割り当てる。
