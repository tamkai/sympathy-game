## 1. Hero Section (First View)
### Visual Concept
*   **Background**: Glassmorphism (すりガラス) + Vibrant Gradients (流体的なグラデーション背景)。
*   **Main Visual**: 巨大な「ピンクの牛」のシルエットが、少し透けたガラスの向こうにおしゃれに佇んでいる。
*   **Motion**: ゆっくりと動く背景、ふわふわと浮遊するUIカード。

### Copywriting
*   **Catchphrase**:
    *   (Main) **「ねぇ、それ『アリ』なの？」**
    *   (Sub) **価値観のズレを楽しむ、共感パーティゲーム。**
*   **CTA Button**:
    *   [Primary] **ルームを作成する (Free)**

## 2. Introduction (Problem/Solution)
### Copywriting
*   **Headline**: **正解なんて、ない。あるのは「共感」だけ。**
*   **Body**:
    *   「目玉焼きに何かける？」
    *   「無人島に一つだけ持っていくなら？」
    *   ありふれた質問でも、みんなの答えは予想外。
    *   Sympathyは、クイズの正解を競うのではなく、**「みんながどう思うか」を予想してシンクロする**ゲームです。
    *   ズレたら負け？いいえ、ズレこそが一番の肴（サカナ）です。

---

## 3. Features (Why Play?)
### Feature 1: Install Free
*   **Title**: **URLを送るだけ。3秒で集合。**
*   **Desc**: アプリのインストールは不要。ブラウザを開けば、そこがパーティ会場。PCでも、スマホでも。

### Feature 2: Aesthetic Design
*   **Title**: **"映える" 画面で、配信もスマートに。**
*   **Desc**: 情報を隠さない、美しいGlassmorphismデザイン。配信画面にそのまま載せても、最高にクール。

### Feature 3: The Pink Cow
*   **Title**: **空気を読まない「ピンクの牛」になれ。**
*   **Desc**: みんなと答えがズレてしまった人は…今日の「ピンクの牛」。ペナルティすら愛おしい、笑いの渦へ。

---

## 4. How to Play (Steps)
1.  **Host**: ルームを作ってURLをシェア。
2.  **Think**: お題に対する「自分の答え」を入力。
3.  **Guess**: みんなの答えの中で「一番多そうなもの」を予想。
4.  **Sympathy!**: 答え合わせ。みんなと揃えば高得点！ズレたら…？
